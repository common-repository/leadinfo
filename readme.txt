This plugin can be used to add the Leadinfo tracking code to a Wordpress site
== Description ==
Identify B2B website visitors. Simple and realtime.

[Leadinfo](https://www.leadinfo.com) enables you to see real-time which companies are visiting your website and what pages they're viewing to increase your sales.
== Installation ==

1. Visit the Plugins Menu in WordPress
2. Search for "Leadinfo"
3. Press "Install"


Configuration Options

1. Visit your Leadinfo Portal and select under 'Trackers' the URL of the website you wish to track.
2. Copy your Leadinfo ID, starting with LI-xxx.
3. Return to WordPress and go to Settings > Leadinfo to paste your Leadinfo ID.
